Glossary

abbr
A tag for marking up abbreviations, acronyms. Requires the "title" attribute for a description of what the acronym stands for.
address
A tag for leaving contact information, usually of the article author. Containing text will be rendered in italic, unless otherwise specified.
aside
A tag for identifying content that is indirectly related to the main content. Usually is used for sidebars and side menus. Doesn't do anything special on its own, requires CSS styling.
code
A tag used for displaying a piece of computer code. Note: HTML still renders normally within it, so to show HTML, you need to remove tags and use HTML Character Codes.
caption
A tag to provide a caption to a table, i.e. allows to set a name, a title or a description to a table that will be shown and by default will be centered above it.
cite
A tag to provide a title of a creative work. By default rendered in italic.
dl, dt, dd
A tag for description lists, described terms and describing descriptions. dl is the tag containing dt followed by dd.
embed
A tag to embed some content. Allows embedding images, html, video, etc.
fieldset
A tag to draw a box around a group of related elements.
figcaption
A tag to provide a caption to a figure element. Think a little description under a picture in a text book from your school days.
legend
A tag to add a name, a caption or a description to a <fieldset> tag.
main
A tag to define the main content of the website. Must not be repeated or be contained within other elements like <article>, <aside>, <footer>, <header>, or <nav>.
mark
A tag to highlight a piece of text with a yellow background (by default).
meter
A tag used for gauges, to visualize a percentage value, e.g. disk usage.
nav
A tag to define a group of navigational links.
optgroup
A tag to group a number of options when using the <select> tag.
option
A tag to define options for the <select> tag.
output
A tag that outputs a result of calculation.
picture
A tag to have more flexibility in displaying images. For example, allows using multiple images for different displays based on their properties, most commonly, width.
progress
A tag to add a progress bar. Needs JS for proper functionality. Not to be confused with <meter>.
samp
A tag used to demonstrate a sample output from a program. Default style is the browser's monospace font. Not as commonly used, CSS being the preferred way to achieve the effect.
section
A tag that marks up a section of the document. Pretty straight-forward.
summary
A tag to define a summary for the content in the <detail> tag. Must be inside of the <detail> tag.
tfoot
A tag to group multiple neighboring cells within a table. Can be used to enable independent scrolling of the various elements of the table. This specific tag refers to the footer of the table.
th
A tag to define header cells. The usage is practically the same as <td>.
thead
See <tfoot> and <tbody>. This specific tag refers to the header of a table.
time
A tag to specify time and/or date. By default doesn't get rendered any differently, but supports the "datetime" attribute to assign a machine-readable format.
track
A tag to add to a <video> or <audio> element to specify a subtitle (or a different kind of text) file/track that will be shown when media is played.
