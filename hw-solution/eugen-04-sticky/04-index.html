dt
Тег dt (description term) создает элемент с названием термина при создании списка dl
dd
Тег dd (description details) создает элемент с описанием термина
dl
Тег dl (description list) создает список
abbr
Тег abbr используется для указания аббревиатуры
addres
Тег address для указания контактной информации. По умолчанию выделяется курсивом.
aside
Тег aside используется для создания боковой панели сайта (sidebar - сайдбар) или вторичного элемента какого-либо блока
code
Тег code - фрагмент кода в тексте. Можно использовать для команд, технических терминов и др.
caption
Тег caption задает заголовок таблицы.
cite
Тег cite указывает источник чего-либо, автора цитаты
fieldset
Тег fieldset- создание группы полей формы. Используется вместе с тегом legend
figcaption
Тег figcaption- заголовок для изображения
legend
Тег legend заголовок для группы формы (fieldset)
mark
Тег mark подсвечивает текст.
output
Тег output используется для вывода результатов вычисления, полученных, как правило, при помощи скрипта
optgroup
Создает группу элементов (option) выпадающего списка select.
option
Элемент выпадающего списка select.
picture
Тег picture- контейнер в котором должен быть один тег img и элементы source с разными источниками и условиями отображения
progress
Тег progress- индикатор выполнения какой-либо задачи
section
Тег section- создание раздела сайта. Обязательно наличие h1-h6 заголовка внутри элемента.
samp
Используется для вывода текста, может пригодится при написании технических статей о каких-либо программах
summary
Указывает заголовок для тега details, по которому можно щелкать для разворачивания/сворачивания информации.
tfoot
Определяет "подвал" таблицы. Включает в себя tr и td.
th
Ячейка строки таблицы, используется для заголовков ячеек.
thead
Определяет "шапку" таблицы.
time
Служит для выделения даты и времени.
track
используется в качестве дочернего элемента для audio> и video, определяя текстовые дорожки
meter
Тег meter служит для вывода значения в известном диапазоне. Может изменять цвет индикатора в зависимости от значения.
menu
Тег menu предназначен для отображения списка пунктов меню.
menuitem
Тег menuitem содержит элементы команды/пункты контекстного меню, которое определяется тегом menu.
keygen
Тег keygen отвечает за генерацию пары ключей (открытый и закрытый), которые используются для шифрования и расшифровки данных форм, а также для создания и проверки цифровой подписи.
main
Основное содержание страницы.
embed
Встраивание файлов или внешних ресурсов на страницу.

Чем отличается b от strong?
Тег strong определяет важность отмеченного текста. Тег b предназначен для выделения текста без придания ему особой важности. Визуально оба тега одинаковы, они выделяют текст полужирным.
